# FacturaIA

AI-powered invoice automation system designed for accounting firms, SMEs, and freelancers in Spain.

## Overview

FacturaIA automatically reads, validates, and organizes Spanish invoices using AI and OCR technologies.

The system is designed to reduce repetitive accounting work, minimize VAT errors, and simplify invoice processing workflows.

Built specifically for:

* Spanish accounting firms
* SMEs
* Freelancers
* Hospitality and retail businesses

---

## Key Features

### Smart OCR + AI Vision

* Reads scanned PDFs
* Reads mobile photos
* Supports JPG, PNG and PDF invoices
* Automatic structured data extraction

### Spanish Tax Validation

* VAT validation (21% / 10% / 4%)
* NIF/CIF verification
* AEAT-oriented workflow
* RD 1619/2012 inspired validation logic

### Workflow Automation

* Parallel invoice processing
* Human review dashboard
* Supplier grouping by NIF/CIF
* Fiscal period alerts (Modelo 303)

### Export & Integration

* Google Sheets export
* Excel (.xlsx) export
* Compatible with:

  * A3
  * Sage
  * Contasol
  * other accounting systems
